# TSGame

> A minimal 2D platformer built with **Godot Engine**.

## 🎮 About
Simple platformer with player movement, enemies, collectibles, and level progression. Built as an MVP/demo.

- **Engine:** Godot 4.x
- **Language:** GDScript
- **Platform:** Windows 10/11
- **Status:** MVP

## ✨ Features
- Player: run, jump, duck, damage system
- Enemies: ground crawler & flying types
- Collectibles: coins, checkpoints, exit goals
- UI: main menu, level select, HUD (health/score)
- Scene transitions with shader effects

## 📁 Structure
```
TSGame/
├── Assets/     # Sprites, fonts, shaders
├── Scenes/     # Game scenes (.tscn) + scripts (.gd)
├── Resources/  # Level data (.tres)
├── Test/       # Unit tests
└── project.godot
```
